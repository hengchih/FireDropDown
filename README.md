FireDropDown
============

When you hover html select tags , this jquery plugin can show option list automatically. In addition, you can also customize list style.


example:

<pre>
 $('#selectA').fireDropDown({
        'color':'#000'
        'background':'#0f0',
        'fontSize':'15px',
        'xShift':0,
        'yShift':0,
        'hover':{
            'background':'#f00'
        }
    });
</pre>
