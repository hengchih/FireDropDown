FireDropDown
============

If you want to customize your html select tag , you can try this jquery plugin.


example:

<pre>
 $('#selectA').fireDropDown({
        'color':'#000'
        'background':'#0f0',
        'fontSize':'15px',
        'xShift':0,
        'yShift':0,
        'hover':{
            'background':'#f00'
        }
    });
</pre>
